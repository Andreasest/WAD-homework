# WAD homework  4
How to run the webapp:
1. Clone this project.
2.  Navigate to the ```/server/``` folder and open ```database.js```.
3.  Enter your postgres password and username in the designated slots (You need to make or use an existing database, but the tables will be created automatically).
4. In the ```/server/``` folder and open the terminal and run ```node server``` or ```nodemon server```.
5. Navigate to the ```/frontend/``` folder and in the terminal run ```npm install``` and ```npm run serve```.
6. The webapp will open at ```localhost:8080``` and the backend will run on ```localhost:3000```.
